# Día 3: El Poder del Orden - Sequenced Collections (LTS Moderno) ⛓️

## 🎯 Contexto Inicial
Hoy experimentarás una de las mejoras más recientes y esperadas de Java: las **Sequenced Collections** de Java 21. Durante décadas, obtener el primer o el último elemento de una lista era un proceso manual y tedioso. Hoy aprenderás a hacerlo de forma profesional y moderna.

## 📚 Conceptos de Base
1.  **Factory Methods (`List.of`, `Map.of`)**: Introducidos en Java 9/11. Permiten crear colecciones con datos iniciales en una sola línea.
2.  **Inmutabilidad**: Las colecciones creadas con `of()` son inmutables. No puedes usar `.add()` ni `.remove()` en ellas.
3.  **Sequenced Collections**: Una nueva interfaz en Java 21 que unifica el manejo de colecciones con un orden predefinido.

---

## 🛠️ Reto Práctico: Navegación de la Lista de Coders
1.  **Factory Methods**: Inicializa una lista inmutable de "Tecnologías Recomendadas" y un mapa inmutable de "Sedes del Corporativo".
2.  **Acceso Secuencial**: Implementa los métodos `getFirst()` y `getLast()` para mostrar el primer y el último empleado de tu lista de forma instantánea.
3.  **Orden Inverso**: Utiliza el método `reversed()` para mostrar la lista completa en orden descendente de registro sin usar algoritmos de ordenamiento manuales.
