# Día 2: Búsqueda Instantánea - HashMap 🗺️

## 🎯 Contexto Inicial
Imagínate que tienes un millón de empleados y necesitas buscar a uno solo por su ID. Si usas una lista, tendrías que revisar uno por uno hasta encontrarlo. ¡Eso es ineficiente! Hoy aprenderás a usar el **HashMap**, la herramienta que permite encontrar datos de forma casi instantánea, sin importar cuántos haya.

## 📚 Conceptos de Base
1.  **Clave-Valor (Key-Value)**: Los mapas guardan datos en pares. La clave (ej: ID del empleado) apunta directamente al valor (ej: Objeto Empleado).
2.  **HashMap**: Una estructura que no tiene un orden predecible, pero es extremadamente rápida para búsquedas.
3.  **Hashing**: La técnica secreta de Java para "mapear" una clave a una posición exacta en la memoria.

## 🛠️ Reto Práctico: El Buscador de Coders
1.  **Indexación**: En tu proyecto industrial, implementa un `HashMap<String, Empleado>` donde la clave sea el ID (String) y el valor sea el objeto `Empleado`.
2.  **Búsqueda Optimizada**: Crea un método que permita al usuario ingresar un ID y que el sistema devuelva los datos del empleado en milisegundos.
3.  **Gestión de Claves**: Asegúrate de que no se puedan agregar dos empleados con la misma clave (ID). Si ocurre, informa al usuario.
