# Día 4: Depuración y Limpieza - El Poder del Filtrado 🧹

## 🎯 Contexto Inicial
Hoy aprenderás a limpiar tus datos de forma profesional. En el pasado, eliminar elementos de una lista mientras la recorrías era un campo de minas para los errores. Hoy aprenderás a usar `removeIf()` para realizar filtrados eficientes y seguros.

## 📚 Conceptos de Base
1.  **Predicados**: Una expresión que devuelve `true` o `false`. Es el corazón del filtrado moderno.
2.  **removeIf()**: El método que permite limpiar una lista completa en una sola línea de código sin usar bucles `for` manuales.
3.  **Inferencia de Tipos (`var`)**: Reforzaremos el uso de `var` en el contexto de colecciones dinámicas para mejorar la legibilidad.

---

## 🛠️ Reto Práctico: Depuración del Talento
1.  **Limpieza Automática**: Implementa el método `removeIf()` para eliminar de la lista a todos los empleados cuyo promedio de desempeño sea inferior a un puntaje mínimo definido por ti.
2.  **Reporte de Sedes**: Utiliza un `HashMap` para contar cuántos empleados hay en cada sede del corporativo y muestra el reporte final.
3.  **Reporte Dinámico**: Utiliza `var` en los bucles de tus reportes para mostrar el total de empleados activos y el promedio salarial del equipo.
