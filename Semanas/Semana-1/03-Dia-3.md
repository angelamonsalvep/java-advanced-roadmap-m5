# Día 3: Lógica de Negocio y Estructura de Paquetes ⚖️

## 🎯 Contexto Inicial
La robustez de un sistema depende de la precisión de su lógica y la claridad de su estructura. Hoy exploraremos el orden en el que Java evalúa las operaciones y cómo organizar diferentes piezas de código para que funcionen como un solo conjunto.

## 📚 Conceptos de Base
1.  **Jerarquía de Operadores**: Java sigue reglas matemáticas estrictas. Conocer el orden de precedencia garantiza que el resultado sea siempre el esperado.
2.  **Operadores Lógicos**: Son el motor de las decisiones. El concepto de "cortocircuito" (`&&` y `||`) permite optimizar la ejecución del programa.
3.  **Classpath**: Es el mecanismo que le permite a Java localizar las clases que componen tu sistema, facilitando la integración de diferentes archivos y librerías.
