# Día 3: Procesamiento Multidimensional y Casting 📊

## 🎯 Contexto Inicial
Hoy aprenderás a procesar datos complejos. Utilizaremos **Matrices** (Arreglos de Arreglos) para almacenar información de rendimiento de múltiples Coders y realizaremos conversiones de datos (Casting) para emitir reportes simplificados.

## 📚 Conceptos de Base
1.  **Matrices (`double[][]`)**: Son tablas de datos con filas y columnas. Se acceden mediante índices anidados: `matriz[fila][columna]`.
2.  **Bucles Anidados**: La forma estándar de recorrer una matriz. El bucle externo recorre las filas y el interno las columnas.
3.  **Casting Explícito**: Convertir un tipo de dato a otro de menor rango (ej: de `double` a `int`). Esto puede provocar **pérdida de precisión**.

1.  **Registro de Notas**: Implementa una matriz de `double` para almacenar 3 trimestres de rendimiento para 3 empleados.
