# Día 2: Captura Dinámica e Inferencia de Tipos (`var`) 📥

## 🎯 Contexto Inicial
Un sistema no es nada sin la entrada del usuario. Hoy aprenderás a capturar datos dinámicos mediante el teclado y a utilizar la inferencia de tipos locales (`var`), una de las características más amadas de Java moderno para limpiar el código.

## 📚 Conceptos de Base
1.  **Scanner**: Una clase de utilidad para leer flujos de datos. Para el teclado, usamos `System.in`.
2.  **Inferencia de Tipos (`var`)**: Introducida en Java 11. Permite que el compilador deduzca el tipo de dato basándose en el valor asignado. El tipo de la variable sigue siendo estático, no cambia.
3.  **Do-While**: Una estructura de bucle que garantiza que el código se ejecute al menos una vez antes de verificar la condición. Ideal para menús.

---

## 🛠️ Reto Práctico: Captura de Información Dinámica
1.  **Menú Persistente**: Utiliza un bucle `do-while` para mantener el sistema activo hasta que el usuario decida salir.
2.  **Uso de `var`**: Declara al menos 5 variables locales dentro de tu bucle (ej: `var id = scanner.nextInt();`) para simplificar la legibilidad del código.
3.  **Validaciones**: Implementa un bloque `if/else` para validar que los datos capturados para los tipos primitivos del sistema (Semana 1) se encuentren en los rangos permitidos.
