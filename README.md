# Corporate Talent Hub - Advanced Java Track 🚀

¡Bienvenido, Coder! Este repositorio es tu hoja de ruta numerada para las próximas 5 semanas. Sigue el orden establecido para asegurar un aprendizaje incremental y coherente.

## 📌 Guía de Inicio Rápido (El Orden de Lectura)

### Paso 1: Configuración del Entorno (Directorio `00-Configuracion-Inicial`)
Antes de codificar, debes preparar tu sistema. Lee los archivos en este orden:
1.  **[01-Ecosistema-de-Versiones.md](00-Configuracion-Inicial/01-Ecosistema-de-Versiones.md)**: Entiende las versiones LTS y distribuciones.
2.  **[02-Guia-de-Instalacion-JDK.md](00-Configuracion-Inicial/02-Guia-de-Instalacion-JDK.md)**: Instala Java 8, 11, 17 y 21.
3.  **[03-Configuracion-del-IDE.md](00-Configuracion-Inicial/03-Configuracion-del-IDE.md)**: Configura IntelliJ, Eclipse, VSC o NetBeans.

### Paso 2: Ruta de Aprendizaje Semanal (Directorio `Semanas/`)
Cada semana tiene una estructura interna fija:
- `00-Soporte-Teorico.md`: Léelo el lunes antes de empezar.
- `01-Dia-1.md` al `04-Dia-4.md`: Retos prácticos diarios.
