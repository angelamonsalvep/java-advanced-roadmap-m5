# Guía de Eclipse IDE (The Classic Veteran) 🪐

Eclipse es el IDE que sostuvo a la industria de Java durante décadas. Sigue siendo fundamental en el sector bancario, financiero y en el desarrollo de aplicaciones empresariales de gran escala (Java EE / Jakarta EE).

## 🛠️ Instalación por Sistema Operativo
Recomendamos el **Eclipse Installer** por su facilidad: [Descargar Eclipse Installer](https://www.eclipse.org/downloads/packages/installer).

*   **Windows**: Ejecuta el `.exe` y selecciona **"Eclipse IDE for Java Developers"**.
*   **Linux**: Descarga el archivo `.tar.gz`, extráelo y ejecuta el instalador.
*   **macOS**: Usa el instalador `.dmg` y arrástralo a tu carpeta de Aplicaciones.

## ⚙️ Configuración Inicial Obligatoria
1.  **Workspaces**: Eclipse organiza todo en "Workspaces". Elige una carpeta de tu disco duro para guardar todos tus proyectos.
2.  **JDK Configuration**: Ve a `Window > Preferences > Java > Installed JREs`. Añade tu **JDK 21** y márcalo como el predeterminado.
3.  **Perspectiva de Java**: Asegúrate de estar en la perspectiva "Java" (arriba a la derecha) para tener las herramientas correctas de navegación.

## 🚀 Características Industriales
- **Ecosistema de Plugins**: Es probablemente el IDE con más extensiones en el mundo a través del **Eclipse Marketplace**.
- **Gestión de Proyectos Gigantes**: Es extremadamente robusto para manejar repositorios con miles de clases.
- **Open Source**: Es 100% gratuito y mantenido por una fundación sin fines de lucro.
