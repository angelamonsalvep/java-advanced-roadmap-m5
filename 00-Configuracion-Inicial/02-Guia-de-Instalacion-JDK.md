# Guía Maestra de Instalación y Configuración ⚙️

Esta sección contiene los lineamientos técnicos para establecer un entorno de desarrollo profesional capaz de manejar múltiples versiones de Java. Este es un recurso **transversal** necesario para completar todas las semanas de la ruta avanzada.

## 🎯 Objetivo de la Configuración
Para realizar las comparaciones entre **Legacy** y **Modern Java**, es imperativo contar con el entorno adecuado. Antes de instalar, te recomendamos leer la **[Guía del Ecosistema de Versiones (LTS, OpenJDK vs Oracle)](JAVA-VERSIONS-EXPLAINED.md)** para entender el porqué de estas versiones.

### 📜 Versiones Legacy (Sistemas Heredados)
1.  **Java 8**: El estándar histórico (Lambdas y Streams).
2.  **Java 11**: La primera transición LTS moderna.

### 🚀 Versiones Modernas (Estándar Actual)
3.  **Java 17**: La base de los Records y el Pattern Matching.
4.  **Java 21**: La versión LTS más avanzada con optimizaciones de concurrencia.
